# Private Message Digest

Use this skill for read-only intake from private message streams.

## Procedure

1. Read only the configured window and source.
2. Classify signals as career-relevant, content seed, operational context, relationship context, or noise.
3. Produce paraphrased summaries rather than raw quotations.
4. Write private digest artifacts and optional notification summaries.
5. Record enough source lineage for private audit without exposing raw third-party text.

## Guardrails

- Do not send WhatsApp or group-chat messages.
- Do not publish raw private chat content.
- Do not treat a group message as permission to use personal details externally.
